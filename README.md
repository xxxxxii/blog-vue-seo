# 使用Vite+Vue3搭建前端SSR应用

## 快速开始
```shell
git clone https://gitee.com/bitem/vite-vue-ssr-demo.git

cd vite-vue-ssr-demo

yarn

yarn dev
```

> 记得修改src/api/config中的接口地址
